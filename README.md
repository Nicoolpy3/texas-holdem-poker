# 🃏 德州扑克 - Texas Hold'em Poker

一款功能完整的网页版德州扑克游戏，采用传统德州扑克规则，支持玩家对战 AI 对手。

## 🎮 游戏特色

- **传统德州扑克规则** - 完整的盲注、下注、摊牌流程
- **智能 AI 对手** - 5 个 AI 玩家，具有不同的决策逻辑
- **完整下注系统** - 支持 Check/Bet/Call/Raise/All-In
- **牌型识别** - 自动判断皇家同花顺、同花顺、四条、葫芦等所有牌型
- **响应式设计** - 适配桌面和移动设备
- **实时游戏日志** - 记录所有游戏动作
- **精美 UI** - 专业扑克桌设计，流畅动画效果

## 📖 游戏规则

### 基本流程

1. **盲注** - 庄家左侧两位玩家分别下小盲和大盲
2. **发底牌** - 每位玩家获得 2 张私密底牌
3. **第一轮下注** - 从大盲左侧玩家开始
4. **翻牌圈** - 发出 3 张公共牌
5. **第二轮下注**
6. **转牌圈** - 发出第 4 张公共牌
7. **第三轮下注**
8. **河牌圈** - 发出第 5 张公共牌
9. **第四轮下注**
10. **摊牌** - 剩余玩家比牌，最大牌型获胜

### 牌型大小（从大到小）

1. 🏆 **皇家同花顺** - A K Q J 10 同花
2. **同花顺** - 五张连续同花牌
3. **四条** - 四张同点数的牌
4. **葫芦** - 三条 + 一对
5. **同花** - 五张同花牌
6. **顺子** - 五张连续牌
7. **三条** - 三张同点数的牌
8. **两对** - 两个对子
9. **一对** - 一个对子
10. **高牌** - 无组合，比最大牌

## 🎯 操作说明

| 操作 | 说明 | 快捷键 |
|------|------|--------|
| 弃牌 (Fold) | 放弃本局 | F |
| 过牌 (Check) | 不下注通过 | Space |
| 跟注 (Call) | 跟随当前下注 | C |
| 加注 (Raise) | 增加下注额 | R |
| 全下 (All-In) | 押上所有筹码 | A |

## 🎨 自定义

### 修改玩家数量

在 `game.js` 的 `initAIPlayers()` 方法中修改：

```javascript
const playerNames = ['你', 'Alex', 'Bob', 'Charlie', 'Diana', 'Eve'];
```

### 调整盲注大小

在 `game.js` 的构造函数中修改：

```javascript
this.smallBlind = 10;
this.bigBlind = 20;
```

### 修改起始筹码

```javascript
const startingChips = 1000; // 修改此值
```
